ABSTRACT

We adopt Pascal's very useful with syntax

DESCRIPTION

        %record = loadrecord($studentID);
        with %record {
		open SPAM, $PreferredSpam;
                print <<SPAM;
        Dear $Name:
                Your tuition is now due.  Please send in a payment
        of at least $Minumum.           

        SPAM
	};

All existing keys of %record hide similarly named variables, if any, within the with block.

These things can nest.

IMPLEMENTATION

within a with block, unqualified variable names must be checked against the with name space before being instantiated.

In concert with named packed structure types serious optimizations could be achieved.
