ABSTRACT

Allow Perl to create serialize both data and code from the core.

DESCRIPTION

The modules Storable and Data::Dumper should be combined to dump data as well as code in text, binary and network order binary form.

IMPLEMENTATION

The immediate need for this is to support unmanned remote devices and lightweight clients where listeners evoke frozen classes upon receiving them.

Other implementation would allow kiosk browsers, for instance, to run any perl from a minimal plugin.

REFERENCES

A sample implementation in Java: websprocket.com
